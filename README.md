## Instalação e Setup:

Clonar o projeto:

```bash
  git clone https://github.com/daniel-neves-dev/rede_yolo.git
```

Abrir a pasta do projeto:

```bash
  cd rede_yolo
```

No terminal digite:

```bash
  pip install numpy
  pip install opencv  
```

No terminal digite:

```bash
  python3 main.py
